Editing

To update a cell, click it, type the new value, and press Return or Tab. Pressing Return moves the input down to the next available cell in the column; pressing Tab moves the input across to the next available cell in the row.

When you update a cell, its background changes to blue.

Updated Cells

Cells with a background cannot be updated.

You can copy and paste values between cells; see 1 Copying and Pasting.

Your changes are saved automatically; there is no need to save them as you go along. If you make a mistake, you can undo changes to individual cells, rows, columns or the entire matrix; see 1 Undoing Changes.
