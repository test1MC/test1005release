Prices Workbench - General Prices Tab

General Prices allows you to maintain the prices that are not specific to a supplier.

 

Field

Description

Co

The company to which this price applies. 0 means this is a global price, which applies across all companies

Resource

The unique code which identifies this resource item.

Sector

The sector to which the price applies.

Development

The number (code) of the development, if this price applies to a single development.

Cat

The cost category to which the price relates.

Gross Price

The price for the selected resource category. The price depends on the rate for the relevant cost category.

Net Price

The price less any discounts.

Per

The unit of measure for the price.

Start

The start date of the price agreement.

End

The end date of the price agreement.

Status

The status of the price (Estimated or Agreed).
