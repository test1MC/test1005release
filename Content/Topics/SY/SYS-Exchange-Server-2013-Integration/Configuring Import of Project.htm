Configuring Import of Project Emails

The steps for configuring import of Project Emails are:

  1. Set up an AppServer on the Unix server (see 1, AppServer)
  2. Create the replication user account in Active Directory (see Active Directory Replication Account)
  3. Create the project public folder tree and folders (section 1, Import of Project Emails)
  4. Install replication software (see 1, Installation)
  5. Configure replication software (see 1, Settings and 1, Command Line Arguments)
  6. Schedule replication (see 1, Scheduling)
