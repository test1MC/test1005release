Dispatch Load Workbench

A new The Dispatch Load Workbench shows the dispatch data files that have been loaded into , and allows you to validate them and create dispatches from them.

The initial browse lists all the files that have been loaded. A link on the Reference column opens a browse that shows the lines.

Dispatch Load Workbench

Load File Lines Browse

The browse of lines in the load file shows line number and the mandatory values pulled from the detail. Column sets are available.

Dispatch Load Workbench - Load File Lines

If a dispatch has been created from the file, a link on the Dispatch No / Line field opens the Dispatch Summary in a new window.

If a line links to a haulage asset, the user is forced to enter a Haulage Charge value on that line.

Actions

The following actions are available on the initial browse of the Dispatch Load Workbench.

Validate File

This checks that the client – that is, the customer or – is valid, and that the item and quantity are available. The checks are the same as applied to dispatch entry.

There is an additional check that there are no null haulage charge values.

Create Dispatches

This creates dispatches as required (one for each customer or ) and adds lines. Haulage charges will create charge lines separate from the main line.
