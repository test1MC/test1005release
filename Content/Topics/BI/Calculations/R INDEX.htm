R-INDEX

Purpose:

 

Returns an integer that indicates the position of the target string within a source string.  In contrast to the INDEX function, R-INDEX performs the search from right to left

Syntax:

 

INDEX(source,target)

 

source:

A character expression.  The string may be a constant, field name, variable, or expression that results in a character value.

 

target:

A character expression that specifies the characters to trim from the string.
