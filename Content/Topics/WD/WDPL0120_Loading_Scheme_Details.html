Loading Details

You can load details from a CSV file, either to load new details or to update existing details.  If a record of the same type already exists for a , the load routine can do one of the following:

  • Overwrite the existing details.
  • Only overwrite the existing details if the date in the file is later than the existing date (to load details of that have been renewed).
  • Not overwrite any existing details.
  • Create a new record of the same type.

To check the file before loading:

  1. Go to Load Data.
  2. Browse for the CSV file that contains the data.
  3. In the Mode field, select Check. 
  4. Detail to show errors in the report.
  5. Choose what you want to happen if there is already a matching record for a .
  6. Click .

    The report lists each in the file, and shows what the result will be when the file is loaded.

  7. If possible, correct any errors in the CSV file before you load it.

To load the data:

  1. Go to Load Data.
  2. Browse for the CSV file that contains the data.
  3. In the Mode field, select Load. 
  4. Choose what you want to happen if there is already a matching record for a .
  5. Click .

File format

The format for the CSV file is:

· Account
· Scheme
· Date
· Policy number
· Provider
· Name on Policy
· Reference
· Limit
· Start Date

with no leading and trailing spaces on the fields.
