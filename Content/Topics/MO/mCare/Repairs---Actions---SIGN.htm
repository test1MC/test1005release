Repairs - Actions - SIGN

Swiping left from the "NOTES" screen will take the to the "SIGN" screen where the signature for the visit can be obtained.

Tapping on the green add (+) button will load the "Signature Capture" screen. The user can select a signatory from the drop down list which will display a list and obtain a signature.

 

Tapping on the signature field, the will be able to capture a signature, After a signature has been captured the operative can also do the following:

  • Cancel - Will disregard the signature that was captures
  • Clear - Remove the signature allowing the to re-capture a signature
  • OK - Saves the signature that has been captured

Once the signature has been captured and the taps "SAVE", the application will prompt the operative to answer a few questions.

If the selects "No", the application will go to the "SIGN" screen where the signatures captured will be listed. However, if the operative selects "Yes" then the application will display the "Survey" for the operative to complete.
