Inspections - Actions - Meter Readings

The selects Meter Readings. To add a meter reading the operative will need to tap onto the green add (+) button

After the has selected to add a meter reading the "Add Meter Reading" screen will be displayed where the user can select the meter reading type. The fields displayed will change in accordance to the meter reading type that has been selected.

 

The keyboard will appear when the user taps on the field that they wish to enter data against. The app will either display a numeric or qwerty keypad depending on the character type of the field.

The user will not be able to proceed unless all mandatory data has been provided.

 

The following details are mandatory (if not completed the user cannot continue):

  • Select meter reading type
  • Serial number
  • Enter reading

The will also be able to add a photo. The user can either capture a photo using the camera from the device or choose an existing one from the gallery.

Notes can also be added against each photo that is captured.

  • The maximum number of photos that can be captured depends on what is set in the OA configuration (CCPHOTOMAXNUMBER)

 

If the wishes to discard the meter reading, They can do so by tapping onto the "X" located in the top left corner of the screen. if/when selected, the following message will be prompted to the operative to confirm their action.

  • Selecting "OK", the Meter Reading will be canceled and the will taken back to the previous screen
  • Selecting "Cancel", the will be able to continue to edit or save their readings.

If the selects "Save", the Meter Reading screen will now display the Meter Reading/s that was captured. The following brief Meter Reading information will be displayed against each reading:

  • Meter Reading Type
  • Serial Number
  • Reading
  • Number of photos captured
  • and the option to delete.

The Operative will also have the option to add another Reading by selecting the green add (+) button. Tapping on the back button
