Bradford Factor Calculation: Bradford Factor Configuration

A new Bradford Factor Configuration page is found under the Human Resources > HR Maintenance menu, as shown in Figure 1: Bradford Factor Configuration Screen. It allows you to predefine an ordered list of time intervals to be used for Bradford Factor calculation, which are displayed in the HR Personnel Workbench.

Bradford Factor Configuration Screen

Field Definitions

Label

The label to be used for the time interval in the Personnel Workbench and the Bradford Factor Report.

Absence Reasons

A comma-separated list of absence reasons to include when calculating the Bradford Factor.

Extensions Allowed Absence Reasons

A comma-separated list of absence reasons where any immediately following absence is taken as an extension of this absence period, rather than as a separate period of absence.

Start Date

The start date of the range for calculating the Bradford Factor. See the Setting Up Start/End Dates Section Section 1.1.1 for more information on setting up this date.

Calced Start Date

The system-generated start date that results from the value in the Start Date column.

End Date

The end date of the range for calculating the Bradford Factor. See the Setting Up Start/End Dates Section Section 1.1.1 for more information on setting up this date.

Calced End Date

The system-generated end date that results from the value in the End Date column.

Dormant

Whether this Bradford Factor is dormant.

Setting Up Start/End Dates

The Start and End Dates set up in Configuration can simply be specific dates, or they can be expressions that evaluate to dates. These expressions can consist of adding/subtracting days, weeks, months, or years to/from a date. You can convert dates to , add/subtract on the result, and finally convert back to a date for the start or end of the or . Any combination of these is permitted as long as the final result is a valid date.
