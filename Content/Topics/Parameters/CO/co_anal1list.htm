CO/ANAL1LIST - Valid Analysis Field 1 codes for this company

A list of valid analysis codes for the first job title analysis set.

Currently job title analysis sets are only used in HR.

This is to facilitate consolidated HR implementations where different have different HR reporting standards and therefore different job titles. Set this as a company-specific parameter and enter a list of the analysis values to allow in this (as set up in Analysis Sets). Only values that match this will appear in the analysis set drop-down.
