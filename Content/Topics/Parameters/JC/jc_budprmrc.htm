JC/BUDPRMRC - Pending Re-measure & Rate Change budget type

The code for the memo data (budget) type that holds pending (unapproved) re-measures and rate changes.

This includes the value of any unapproved which are a re-measure or rate change type. If this is blank then BUDRMRC will be used instead.

If JC/BUDPRMRC is not blank, the Budgets browsers in Status show a column that allows a drill down to show all unapproved which are a re-measure or rate change type and have a resource with the selected cost code/WBS code/section/activity.
