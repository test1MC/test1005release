PR/PR-PAYSUMMARYFILE - Payment Summary Output file [AUS]

The folder and file name of the EOY ATO Payment Summary Output file (on the server). (To make it appear in "My files", the folder name should begin with $BASE/var/home/[USER]/ )

The file name can contain tokens in square brackets that will be replaced when creating the file. This is necessary if you are creating more that one file in a run.
The tokens that can be used are:
[KCO] Replaced with the number of the logged-in .
[USER] Replaced with the user ID of the current user.
