HS/WFDMTYPE - The Workflow Responsibility Type for roles managed in Development Maintenance

The workflow responsibility type that is used for roles whose workflows are managed at level.

If a Workflow Responsibility Type has been entered in WFDMTYPE, a Responsibilities tab is shown in Maintenance. This shows a list of the Workflow roles held in WFDMRLES and allows staff to define the user(s) who are responsible for each role in a workflow.
